- 스트레스 테스트를 진행하기 전에 진행
  - 로드 테스트를 통과한다면 스트레스 테스트를 수행
- 로드 테스트는 시스템을 고의로 고장내는 테스트는 아님

- 인프라가 변경된 후에는 어떤 경우라도 로드 테스트를 진행하는 것이 좋음
